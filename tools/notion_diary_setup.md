# Notion 日記データベース セットアップ手順

じぶんリリースノートの下書き生成に使う日記データベースの作成手順です。

## 1. データベースを作成する

1. Notionで任意のページを開く（または新規ページを作成）
2. `/database` と入力 → 「データベース - フルページ」を選択
3. データベース名を「日記」にする

## 2. プロパティを設定する

デフォルトの「名前」プロパティはそのまま **タイトル** として使います。
以下のプロパティを追加してください。

### 日付（Date型）

1. 「+」ボタンでプロパティを追加
2. 名前: `日付`
3. 型: `日付`

### 感情（Select型）

1. プロパティを追加
2. 名前: `感情`
3. 型: `セレクト`
4. 以下のオプションを追加（色も設定）:

| オプション名 | 色     |
| ------------ | ------ |
| いい調子     | Green  |
| ふつう       | Blue   |
| イライラ     | Red    |
| 不安         | Yellow |
| 憂鬱         | Purple |

### カテゴリ（Multi-select型）

1. プロパティを追加
2. 名前: `カテゴリ`
3. 型: `マルチセレクト`
4. 以下のオプションを追加:

- `仕事`
- `専門スキル`
- `コミュニティ`
- `交友`
- `健康`
- `生活`

> カテゴリは任意入力です。毎日つけなくてもOKですが、つけておくと月末のKPT分類が楽になります。

## 3. ビューを設定する

### デフォルトビュー: カレンダー

1. ビュータブの「+」 → 「カレンダー」を選択
2. 名前: `カレンダー`
3. 日付プロパティで表示
4. 日々の記入にはこのビューを使う

### 月次テーブルビュー

1. ビュータブの「+」 → 「テーブル」を選択
2. 名前: `月次テーブル`
3. ソート: 日付の昇順
4. 表示カラム: タイトル, 日付, 感情, カテゴリ
5. **月末のエクスポートにはこのビューを使う**

### 感情ボードビュー（任意）

1. ビュータブの「+」 → 「ボード」を選択
2. 名前: `感情ボード`
3. グループ化: 感情プロパティ
4. その月の感情の分布が一目でわかる

## 4. テンプレートを作成する（任意）

1. データベース右上の「新規」ボタンの「▼」→「新規テンプレート」
2. テンプレート名: `日記テンプレート`
3. プロパティのデフォルト値:
   - 日付: 「今日」
   - 感情: （空欄のまま）
4. ページ本体に以下を入れておくと書きやすい:

```
## 今日のこと

## メモ
```

## 5. 日記の書き方

毎日（または書きたい日に）以下を記入します:

- **タイトル**: その日のひとこと（例: 「同僚と写真散歩した」「体調不良で早退」）
- **日付**: 当日の日付
- **感情**: 5つから1つ選ぶ
- **カテゴリ**: 該当するものを選ぶ（複数可、任意）
- **ページ本体**: 自由に書く（文字数制限なし）

## 6. 月末のエクスポート手順

月末にじぶんリリースノートの下書きを作るとき:

1. 月次テーブルビューを開く
2. フィルタで当月の日付範囲を設定
3. 全エントリを選択（左端のチェックボックス）
4. 右上の「…」メニュー → 「エクスポート」
5. フォーマット: 「Markdown & CSV」を選択
6. エクスポートされたZIPを展開
7. 展開したフォルダをClaude Codeに渡す

```
# 例: Claude Codeへの依頼
cd ~/dev/mom0tomo.github.io
# → Claude Codeに以下のように依頼:
#
# ~/Downloads/diary_202601/ にある今月の日記データから、
# じぶんリリースノートの下書きを作ってください。
# 歩数: 平均7200歩/日
# 本（読了）: ○○○
# 本（途中）: ○○○
```
